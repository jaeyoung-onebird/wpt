# WorkProof Token (WPT) Tokenomics

**버전**: 1.0
**작성일**: 2026-02-04

---

## 💡 WPT 개념

**WPT = 플랫폼 이용권/연료**

- 블록체인 토큰이 아닌 **플랫폼 포인트**
- 근무하면 획득, 기능 사용하면 소비
- 기업이 더 많이 소각 → 디플레이션 구조
- 게임화 + 실용성 결합

---

## 📊 토큰 발행 정책

### 초기 발행
- **총 발행량**: 무제한 (Earn에 의해 생성)
- **초기 보상 풀**: 100,000 WPT (시드)
- **Admin 보유**: 50,000 WPT (이벤트/보상용)

### 발행 메커니즘
- **Earn**: 근무자 활동으로 생성
- **Mint**: Admin 수동 발행 (이벤트/보상)
- **제한**: 하루 최대 500 WPT/인

## 💰 Earn - WPT 획득

### 출퇴근 보상

| 활동 | 금액 | 조건 | 빈도 |
|------|------|------|------|
| 출근 성공 | +10 WPT | GPS/코드 인증 | 1회/일 |
| 퇴근 성공 | +10 WPT | 정상 퇴근 | 1회/일 |
| 근무시간 보너스 | +5 WPT/시간 | 실제 근무 | 시간당 |
| 지각 페널티 | -5 WPT | 30분 이상 지각 | - |
| 조기퇴근 페널티 | -10 WPT | 예정보다 1시간 이상 조기 | - |

**예시**:
```
출근: +10 WPT
근무 8시간: +40 WPT (5×8)
퇴근: +10 WPT
──────────────
총 획득: 60 WPT/일
```

## ⚖️ 경제 균형

### 목표 비율

```
기업 소각량 : 근무자 획득량 = 2 : 1
```

### 월간 예상 (근무자 1,000명 기준)

**Earn (획득)**:
```
평균 근무자: 월 20일 근무
일 60 WPT × 20일 = 1,200 WPT/월/인
전체: 1,200 × 1,000 = 1,200,000 WPT/월
```

**Burn (소각)**:
```
기업 100개 × 월 25,000 WPT = 2,500,000 WPT/월
(AI 추천, 리포트, 광고 등)
```

**결과**:
```
발행: 1,200,000 WPT
소각: 2,500,000 WPT
─────────────────────
순 감소: -1,300,000 WPT (디플레이션)
```

### 인플레이션 방지 메커니즘

1. **기업 사용 의무화**
   - 급여명세서: 필수 (10 WPT)
   - 행사보고서: 필수 (10 WPT)
   - AI 추천: 강력 권장

2. **Spend 유도**
   - NFT 배지 수집 욕구
   - 프로필 부스트 혜택
   - 스테이킹 고수익

3. **발행 제한**
   - 일 최대 500 WPT/인
   - 월 최대 15,000 WPT/인
